# EDA & Business Intelligence Project

## Project Overview

This project performs Exploratory Data Analysis (EDA), SQL-based business analysis, KPI identification, and dashboard development using the Sample Superstore retail dataset.

## Objectives

* Clean and preprocess retail sales data
* Perform descriptive and exploratory data analysis
* Answer business questions using SQL
* Identify key performance indicators (KPIs)
* Create visualizations and business insights
* Build a Power BI dashboard
* Generate a final EDA report

* West region achieved the strongest sales performance.
